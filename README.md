# Marketing Campaign Analysis & Customer Behavior Prediction
## Overview
This project focuses on analyzing customer behavior and optimizing marketing strategies using data analytics and machine learning techniques.

## Objective
To identify patterns in customer data and build predictive models that improve marketing decision-making.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Tableau (for visualization)

## Machine Learning Models
- Logistic Regression (AUC: 0.87)
- Decision Trees
- K-Means Clustering

## Data Analysis
- Analyzed dataset with 2000+ observations
- Identified behavioral trends (screen time, stress, sleep)
- Segmented customers based on behavior

## Visualization
- Built interactive Tableau dashboards
- Visualized campaign performance and customer insights

## Key Insights
- Identified customer segments for targeted marketing
- Improved understanding of behavioral patterns
- Supported data-driven marketing decisions

## Research Relevance
This project reflects my interest in Applied AI, particularly in using machine learning to solve real-world business and marketing problems.

## Future Work

- Deploy models for real-time predictions
